## Development

1. **Install node/npm/yarn.** The app has been tested with node v12.

2. **Clone and install dependencies**:
```shell
$ git clone https://github.com/alkafinance/breathe.sh.git breathe-sh
$ cd breathe-sh
$ yarn
```

3. **Run the app**:
```shell
$ yarn dev
```
